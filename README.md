# 📚 Online Bookstore Database & Sales Analytics: From Inventory to Insights

This project demonstrates the design and analysis of a relational database for an **Online Bookstore** using SQL. The goal of the project is to build a structured database system and extract meaningful business insights from transactional sales data.

The database stores information about **books, customers, and orders**, allowing analysis of customer purchasing behavior, sales performance, and inventory trends.

---

## 🎯 Project Objectives

* Design a **relational database schema** for an online bookstore.
* Perform **data cleaning and preparation** using SQL.
* Write SQL queries to answer **real-world business questions**.
* Generate insights about **sales performance, customer trends, and inventory management**.

---

## 🗂️ Dataset

The project uses the following tables:

**Books Table**

* Book ID
* Title
* Author
* Genre
* Published Year
* Price
* Stock

**Customers Table**

* Customer ID
* Name
* Email
* Phone
* City
* Country

**Orders Table**

* Order ID
* Customer ID
* Book ID
* Order Date
* Quantity
* Total Amount
